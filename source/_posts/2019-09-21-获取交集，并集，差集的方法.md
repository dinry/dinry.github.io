# 交集
```js```
#方法一:
a=[2,3,4,5]
b=[2,5,8]
tmp = [val for val in a if val in b]
print tmp
#[2, 5]

#方法二
print list(set(a).intersection(set(b)))

#方法二比方法一快很多！
```js```

# 并集
```js```
print list(set(a).union(set(b)))
```js```
# 获取两个 list 的差集
```js```
print list(set(b).difference(set(a))) # b中有而a中没有的      非常高效！
```js```
